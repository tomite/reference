リテラルを使用する

リテラルの種類と記述方法
リテラルの種類　　　生成される値の型　　記述方法　　　　　　　　　　　　　例
整数(10進数)        int                 数字を列挙する        　　　　　　100
整数(16進数)        int                 数値の先頭に0xまたは0Xをつける    0xa1
整数(8進数)         int                 数値の先頭に0をつける             075
整数(2進数)         int                 数値の先頭に0bをつける(Java7以降) 0b11
整数                long                整数値の最後にlまたはLをつける    100L
浮動小数点数        float               小数の最後にfまたはFをつける      1.23f
浮動小数点数        double              小数                              1.23
浮動小数点数        double              指数表現                         123e-10
文字                char                シングルクオートで文字を囲む      'a'
文字列              String              ダブルクオートで文字列を囲む    "hello"
真偽値              boolean             true, false                      true
null値              null型              null                             null

Java7以降は数値リテラルの中にアンダースコア(_)を入れることができる。
